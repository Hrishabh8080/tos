# Video Files Directory

Place your hero video files here:

- `products-hero.mp4` - Main video file (MP4 format)
- `products-hero.webm` - Optional WebM format for better browser support

## Video Requirements:
- **Format**: MP4 (H.264 codec) or WebM
- **Recommended Size**: 1920x1080 or 1280x720
- **File Size**: Keep under 10MB for better performance
- **Duration**: 10-30 seconds (will loop automatically)

## How to Add Video:
1. Place your video file(s) in this directory
2. Name them exactly: `products-hero.mp4` and/or `products-hero.webm`
3. The video will automatically play on the products page

## Note:
If no video file is found, the hero section will display a gradient background instead.
